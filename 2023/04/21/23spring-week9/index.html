开发

javaWeb项目通常打包成war包（设置打包方式是在pom.xml文件中）

  • pom.xml文件主要是配置项目相关的依赖
  • web.xml文件主要配置web相关

springMVC默认获取的是queryString格式（请求路径）的参数

如果在参数中加入@RequestBody注解，则说明参数请求的参数使用Json格式来传递的

@PutMapping("/user")
public ResponseResult updateUser(@RequestBody User user){
    userService.updateUser(user);
    return new ResponseResult(200,"操作成功");
}

加上@PathVariable注解，则说明参数是从路径中获取（RestFul风格

配置拦截器的拦截路径

使用return跳转到一个页面，跳转的方法是请求转发而不是重定向（请求转发浏览器地址栏的地址是不变的

代码随想录

贪心

t1

我的做法是最原始的模拟，先计算出每个出现的区间，在去计算分割

更优雅的做法是，只需要记录每个字符的最靠右的坐标，一次遍历字符串s，当出现下标和最靠右下标重合时，即为一个分界点

动态规划

动态规划是由前一个状态推导的，贪心是直接局部选最优

动态规划的debug可以利用打印dp数组

动态规划问题五步曲：

  1. 确定dp数组（dp table）以及下标的含义
  2. 确定递推公式
  3. dp数组如何初始化
  4. 确定遍历顺序
  5. 举例推导dp数组

确定dp数组以及下标的含义=>递推公式=>初始化=>遍历顺序
